# image-resizer
 Offline image resizer using Node.JS.
 
 Packages used: FileSystem(fs), Path and Sharp.
 
 To test it, run "node resizer img/logo.png" on node terminal, in project folder.
 
 Node v12.19.0
 npm v6.14.8
